# CyberFlower Language｜未来花语

这是一个面向未来情感交流场景的交互艺术编程原型。系统将情绪类型、强度、持续时间、开放程度、声音输入与鼠标距离映射为数字花朵的形态、色彩、运动节奏、粒子轨迹和光效。

## 打开方式

1. Windows 双击 `启动项目.bat`。
2. 或直接打开 `index.html`。
3. 若浏览器限制麦克风权限，可使用 VS Code 的 Live Server 打开。

## 成品功能

- 8 种情绪数字花：喜悦、孤独、焦虑、平静、思念、愤怒、治愈、期待
- 情绪强度、持续时间、开放程度参数
- 鼠标距离控制花朵开放
- 麦克风音量控制粒子与光晕
- 未来花语文本生成
- 本地花语档案
- 花语卡 PNG 导出
